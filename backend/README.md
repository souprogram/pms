# PMS Backend

This is the backend API for the PMS application.

## Local Development

```bash
# Install dependencies
npm install

# Run in development mode
npm run dev

# Run tests
npm test
```

## Deployment

### Prerequisites

- Node.js v20.18.3 or compatible version
- Environment variables set up (see .env.example)

### Build for Production

```bash
npm run deploy
```
